Aerondo Documentation
Welcome! We are excited that you want to learn more about Aerondo and the features of the platform. On the following sites you find helpful information about the features offered by Aerondo and hints how to use them.
What is Aerondo
Aerondo is your ideal partner when it is about managment of data originating from general aviation. Whether you want to manage your personal information, the data of an aircraft, an aerodrome or an organisation (a company or an association), alone or together with other people, Aerondo has tools that make your life easier. Aerondo is a web application built on the latest web and cloud technologies. This means you don't need to install something but can use Aerondo on any device you want to. You just need a browser to sign up for a new account and to log in. Then you are all set and ready to use Aerondo. Using Aerondo means you have your data and the management tools always with you on your smartphone and available on every other device you want to use to mange your general aviation data.
Get started with Aerondo
To get started with Aerondo you neet to sign up for a new account. Currently the sign up is inactive because the Aerondo project is just started and only the frist features are released for use. This means potential users can only use a limited feature set in their daily general aviation activity. If you are interested to try aerondo in its current state get in touch with us by sending us a mail with some information what features you are looking for and what data you want to manage. We will then create an account for you.
